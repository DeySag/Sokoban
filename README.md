To understand what the original game is about, how it is played and what are its variants, refer to the following link:

https://alonso-del-arte.github.io/sokoban-ascii/index.html

The approaches here include two specific variants of the A* algorithm on the single box single goal problem. One is the Move-Optimal approach where it finds the optimal path after searching a significant number of nodes (but less than that of BFS) while the Time-Optimal approach finds a slightly less optimal path in significantly less node traversals.
